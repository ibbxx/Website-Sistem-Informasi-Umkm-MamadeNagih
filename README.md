# Mamade Nagih - Keripik Pisang Premium

Mamade Nagih adalah produsen keripik pisang premium dari bahan lokal Enrekang. Renyah, halal, dengan varian rasa kekinian. Cocok untuk camilan, oleh-oleh, dan peluang usaha kemitraan.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Customizing Images

All images are located in the `public/images` directory. To replace an image, simply upload a new file with the same name to that folder.

For example, to change the hero image:
1. Prepare your new image.
2. Rename it to `hero-product.jpg`.
3. Replace the file in `public/images/hero-product.jpg`.

You can also manage image paths in `src/config/images.ts`.

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
